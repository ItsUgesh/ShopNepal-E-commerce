🛍️ ShopNepal – E-commerce Clothing Platform

ShopNepal is a modern WordPress-based e-commerce website for clothing,
designed to provide a smooth shopping experience with essential features
like product browsing, cart management, user authentication, and order tracking.

Built using Elementor and WooCommerce, the platform focuses on usability,
responsive design, and real-world functionality for online shopping in Nepal.

✨ Features

🛒 Add to cart and manage products easily

🔐 User authentication (Login & Registration system)

📦 Order tracking functionality

🧾 Full e-commerce workflow powered by WooCommerce

📱 Fully responsive design for mobile, tablet, and desktop

⚡ Clean and user-friendly UI built with Elementor
